    mixins

    devices

    @mixin devices($breakpoint: $breakpoint-list) { ... }

    Description

    This mixin will be create responsive layout for page or components use by media at-rule.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Reference of breakpoint component. If $breakpoint without use then this mixin will be generate all breakpoints depend of $breakpoint-list. If want to numerical value for breakpoint, value suffix must be px.

    		List or String$breakpoint-list

    Example

    devices mixin

    .page {
  background-color: red;
  /** When parameter without use then this mixin will be generate for all breakpoints depends of $breakpoint variable values. */
  @include devices(mobile) {
    background-color: green;
  }
}

/**
.page {
 background-color: red;

media only screen and (588px > width) {
 .page {
   background-color: green;
 }
 */

    Requires

    • [variable] desktop-m
    • [variable] desktop-m
    • [variable] desktop
    • [variable] desktop
    • [variable] tablet
    • [variable] tablet
    • [variable] tablet
    • [variable] mobile
    • [variable] mobile
    • [variable] mobile
    • [variable] breakpoint-list — If not passed any value in mixin with this variable will be use as default

    component-query

    @mixin component-query($breakpoint: $breakpoint-list) { ... }

    Description

    This mixin will be generate component style based on width of component.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Reference of breakpoint component. If $breakpoint without use then this mixin will be generate all breakpoints depend of $breakpoint-list. If want to numerical value for breakpoint, value suffix must be px.

    		List or String$breakpoint-list

    Example

    component-query mixin

    .component {
  background-color: red;
    /** When parameter without use then this mixin will be generate for all breakpoints depends of $breakpoint variable values. */
  @include component-query(mobile) {
    display: grid;
  }
}

/**
 .component {
  background-color: red;
}
 @container mobile 588px>width {
  .component {
    display: grid;
  }
}
 */
